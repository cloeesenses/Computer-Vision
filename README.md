# Tema 5: Visión por computadora

En este tema desarrollé 3 proyectos de visión por computadora usando reconocimiento facial.

Cada carpeta corresponde a un proyecto diferente y cada proyecto cuenta con su propio `README.md` y su propio archivo `requirements.txt`, donde se explican las instrucciones de instalación, ejecución y dependencias necesarias.

## Proyectos desarrollados

1. **Sistema de asistencia automatizada con reconocimiento facial**  
   Registra la asistencia de una persona cuando su rostro coincide con un rostro previamente registrado.

2. **Sistema de inicio de sesión seguro con verificación facial**  
   Permite o niega el acceso a un sistema comparando el rostro de un usuario autorizado con el rostro de quien intenta iniciar sesión.

3. **Sistema de reconocimiento facial para hogares inteligentes**  
   Registra perfiles de usuarios, guarda su rostro y canción favorita, y permite el acceso al hogar si el rostro es reconocido.

## Nota

Para ejecutar cada proyecto, se debe entrar a su carpeta correspondiente y seguir las instrucciones de su propio `README.md`.
